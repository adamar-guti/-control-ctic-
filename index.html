🔑 Datos de acceso

📋 Pasos para configurar

  1. Crea el repo control-ctic en GitHub como Público
  2. Ve a github.com/settings/tokens/new
  3. Nombre: control-ctic · Expiration: No expiration
  4. Activa el permiso repo (acceso completo)
  5. Copia el token y pégalo arriba
  6. Clic en Inicializar repo (crea las carpetas necesarias)
  7. Activa GitHub Pages: Settings → Pages → Branch main → /root
